O que é GIT ?
O Git é um sistema de controle de versão distribuído, ou seja, um sistema que permite rastrear alterações em arquivos ao longo do tempo, facilitando a colaboração em projetos de desenvolvimento de software e a recuperação de versões anteriores. 

#Comandos do GIT
O que é GIT STATUS ?
O comando git status no Git exibe o estado do seu diretório de trabalho e da área de staging (ou staging area). Em resumo, ele mostra quais arquivos foram modificados, adicionados ou removidos, e se estão ou não preparados para serem incluídos no próximo commit. 

O que é GIT COMMIT ?
Um "commit" no Git é um registro de alterações feitas em um repositório, que captura um ponto específico no histórico do projeto. É como um snapshot do código no momento do commit, permitindo rastrear as mudanças e retornar a um estado anterior, se necessário.

O que é GIT PUSH ?
O git push é um comando Git que serve para enviar as alterações (commits) de um repositório local para um repositório remoto. Em outras palavras, é a forma como você compartilha o seu código com outros colaboradores ou faz um backup do seu projeto em um servidor remoto.
